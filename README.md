# CodeAlpha_HotelReservationSystem
A console-based Hotel Reservation System developed in Java that allows users to search available rooms, book rooms, cancel reservations, and view booking details. This project demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, and collections.

Features
View available hotel rooms
Book rooms with customer details
Cancel existing reservations
Display current bookings
Room categorization (Standard, Deluxe, Suite)
Payment simulation
Console-based user interface

Technologies Used
Java
Object-Oriented Programming (OOP)
ArrayList
Scanner Class
Collections Framework

Concepts Covered
Classes & Objects
Constructors
ArrayList
Encapsulation
Menu-driven Programming
Conditional Statements
Loops
File Structure using Multiple Classes
